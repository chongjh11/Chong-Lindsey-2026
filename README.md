This is the supplementary data for Chong and Lindsey, in review in Geophysical Research Letters. This includes the scripts and examples (from Pylith) to perform inversion on variable material properties of a subduction zone.

**MATLAB folder**
- Models_final_slab_thickness_2d (Homogeneous Green's function)
- Models_final_slab_varymatwedge_elastic_2d (Variable material properties/Heterogeneous Green's function)
- Scripts (necessary scripts needed to run the inversion and plot figures)

  
**Pylith folder**
- Includes the necessary scripts to recreate the mesh, run Pylith to get the Green's function for both Homogeneous and Heterogeneous models
  - Consists of the config file and spatialdb to run the models (Green's function).
